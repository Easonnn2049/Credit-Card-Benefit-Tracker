# Project Instructions

This project is a local-first Python + Streamlit credit card benefit tracker.

Before making any UI, UX, visual design, information architecture, dashboard, interaction, or product-flow changes, read and follow `PRODUCT_DESIGN_BRIEF.md`.

Keep implementation simple and maintainable:

- Stay in Python + Streamlit.
- Preserve the local CSV/Excel-based data workflow.
- Do not add cloud sync, authentication, bank integrations, scraping, or complex frontend frameworks.
- Prefer incremental changes over rewrites.
- Keep the interface scan-friendly, action-oriented, and beginner-friendly.
